# 🎾 노원 마들 테니스장 예약 크롬 익스텐션

노원 마들 테니스장 예약 사이트에서 날짜, 코트, 시간을 선택하면 예약을 자동으로 실행해주는 크롬 익스텐션입니다.

---

## 설치 방법

### 1단계 — 익스텐션 파일 준비

이 저장소를 클론하거나 ZIP으로 다운로드합니다.

```bash
git clone <저장소 URL>
```

또는 GitHub 페이지에서 **Code → Download ZIP** 후 압축을 해제합니다.

### 2단계 — 크롬에 로드

1. 크롬 브라우저 주소창에 `chrome://extensions` 입력 후 접속
2. 우측 상단 **개발자 모드** 토글을 **ON**으로 변경
3. **"압축 해제된 확장 프로그램 로드"** 버튼 클릭
4. `nowon-tennis-extension/` 폴더를 선택

> 정상 로드되면 크롬 툴바에 🎾 아이콘이 나타납니다.

---

## 사용 방법

### 1. 예약 홈으로 이동

1. 크롬 툴바에서 🎾 아이콘 클릭
2. **"예약 홈으로 이동"** 버튼 클릭
   - 현재 탭이 예약 사이트(`reservation.nowonsc.kr`)이면 현재 탭에서 이동
   - 아니면 새 탭으로 열림

### 2. 예약 실행

1. 예약 사이트(`reservation.nowonsc.kr`)가 열려 있는 상태에서 🎾 아이콘 클릭
2. **날짜** 선택 (오늘 이후 날짜만 선택 가능)
3. **코트 번호** 선택 (1~9번, 복수 선택 가능)
4. **시간** 선택 (06:00~22:00, 복수 선택 가능)
5. 하단 **선택 현황** 확인 후 **"예약하기"** 버튼 클릭

### 선택 현황 카운터

팝업 하단에 실시간으로 예약 건수가 표시됩니다.

```
선택 현황: 코트 2개 × 시간 2개 = 4건
```

- ✅ 4건 이하: 초록색 표시
- ⚠️ 4건 초과: 빨간색 경고 표시

---

## 제약사항

| 항목 | 내용 |
|------|------|
| 날짜 | 단일 선택, 오늘 이후만 가능 |
| 코트 | 1~9번, 복수 선택 가능 |
| 시간 | 06:00~22:00 (1시간 단위), 복수 선택 가능 |
| 최대 예약 건수 | **코트 수 × 시간 수 ≤ 4건** |
| 동작 사이트 | `https://reservation.nowonsc.kr/*` |

> 예: 코트 2개 + 시간 2개 = 4건 (가능) / 코트 2개 + 시간 3개 = 6건 (불가)

## 오류 메시지 안내

| 메시지 | 원인 및 해결 방법 |
|--------|------------------|
| 날짜를 선택해주세요. | 날짜 입력란이 비어 있음 → 날짜 선택 후 재시도 |

| ❌ 예약 실패 | 예약 사이트에서 오류 발생 → 사이트 로그인 여부 확인 |
